## Application Architecture

The Flask ToDo App follows a clean, modular structure:

```python app/__init__.py
from flask import Flask
from flask_sqlalchemy import SQLAlchemy
from .config import load_app_config
from .db_utils import get_engine_type

db = SQLAlchemy()

def create_app():
    app = Flask(__name__)
    
    config = load_app_config()
    app.config['SQLALCHEMY_DATABASE_URI'] = config['SQLALCHEMY_DATABASE_URI']
    app.config['SQLALCHEMY_TRACK_MODIFICATIONS'] = False
    app.config['style'] = config.get('style', {})

    @app.context_processor
    def inject_engine():
        return dict(engine=get_engine_type())
    
    db.init_app(app)

    with app.app_context():
        from . import routes
        from .models import Task
        db.create_all()

    return app
```

<Note>
  The application uses the **factory pattern** with `create_app()`, making it easy to test and deploy in different environments.
</Note>

## Simple Data Model

At the core is a straightforward Task model:

```python app/models.py
from . import db

class Task(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    name = db.Column(db.String(255), nullable=False)
    completed = db.Column(db.Boolean, default=False)
    locked = db.Column(db.Boolean, default=False)
```
